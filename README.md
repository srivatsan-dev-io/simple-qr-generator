# Simple QR Code Generator

A simple QR Code Generator written in HTML, CSS and Vanilla Javascript that converts any text or URL into a QR code using a free API.

---

## Features

- Generates QR codes from text or URLs.
- Instant display of the QR code.
- Simple to use.

---

## Usage 

1. Open `index.html` in a browser.
2. Enter a text or URL.
3. Click **Generate QR Code**.
4. The QR Code is generated.
